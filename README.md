# Corvus

Exploratory pattern analysis engine. Detects routines, emerging interests, anomalies, stalled threads, and cross-domain opportunities from the knowledge graph and accumulated activity signals.

## Repository Structure

- `ocas-corvus/` - Skill package source
- `spec/` - Specification and build documentation

## Installation

This skill package is available on GitHub. To use it, clone this repository and follow the installation instructions in your skill management system.
